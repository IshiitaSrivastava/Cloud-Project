# Cloud Voting System

Simple cloud-deployable voting system for college project.

## Structure
- backend/ — Flask backend (API + JWT auth + SQLite)
- frontend/ — Static HTML/JS UI
- tests/ — basic test
- Dockerfile, docker-compose.yml — for containerized run
